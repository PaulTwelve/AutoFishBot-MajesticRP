# ***Спасибо за выбор AutoFishBot от PaulTwelve***

## Важная информация

- Почему бот бесплатный?  
Бота писал для себя с целью повысить навыки и решил им поделиться чтобы наслаждаться рыбалкой вместе.

- Бот написан на языке AutoHotkey - языке скриптов, а это значит что он не изменяет игру, не детектится анти-читами и не может быть забанен, а также в нём есть случайные задержки, поэтому вы можете спокойно им пользоваться и не бояться бана или что кто-то вас заподозрит.

- Если на ПК установлен антивирус помимо Windows Defender - отключите его.

- Перед использованием распакуйте в удобное место.

- Пожалуйста не переименовывайте скрипт.

- Больше года использования, более 300 пользователей и ни одного бана.

- При наличии 2-ух мониторов - используйте левый

- Открыть окно программы: ```Ctrl + Shift + 1```

## Начало рыбалки
  
1. Запускаете скрипт AutoFishBot.exe
2. Задаёте клавиши и нажимаете «Сохранить»
3. Встаёте в зону закидывания удочки
4. Нажимаете на назначенную кнопку «Вкл/Выкл»
  
## Дополнительная информация о скрипте

- После переполнения инвентаря, ошибке или окончания наживки - скрипт автоматически остановится.
- Рыбача от 1-ого лица, при наличии очков - их нужно снять.
- Скрипт активно обновляется, добавляются новые функции и улучшается пользовательский интерфейс для более комфортного опыта его использования.  
Обновляется автоматически.
- Получив активацию - бота нужно перезапустить.

## Если бот не работает
Проверьте чтобы ваша система может выводить FullHD
  
1. ПКМ по рабочему столу  
2. ```«Параметры экрана»```  
3. Разрешение дисплея должно быть ```«1920 x 1080»```  

### Запуск от имени администратора

Для удобства вы можете настроить автоматический запуск с правами администратора:
  
1. Открываете ```«Свойства»``` программы (ПКМ)
2. Переходите в раздел ```«Совместимость»```
3. Поставьте галочку рядом с ```«Запускать эту программу от имени администратора»```

### Windows 11

Перед запуском бота на Windows 11 нужно:
  
1. Открываете ```«Свойства»``` программы (ПКМ)
2. Переходите в раздел ```«Совместимость»```
3. Поставьте галочку рядом с ```«Запустите эту программу в режиме совместимости для:»``` ```Windows 8```
